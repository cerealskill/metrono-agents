# ROSTER.md

## Team
Content & Marketing Team

## Objective
Research, produce, and distribute high-quality content that drives awareness and engagement. The team follows a research-then-execution pattern — strategy comes first, production second.

## Members

| Agent | Role | Responsibility |
|-------|------|----------------|
| `marketing` | Content Strategist | Owns editorial calendar, channel strategy, and content briefs |
| `doc` | Technical Writer | Produces long-form articles, documentation, and deep-dive content |

## Interaction Model

Strategist briefs → Writer drafts → Strategist reviews → Strategist distributes.

```
┌──────────────────────┐      ┌──────────────────┐
│  Content Strategist  │─────>│ Technical Writer │
│  (brief + direction) │      │  (draft content) │
└──────────────────────┘      └────────┬─────────┘
                                        │ draft ready
                                        v
                              ┌──────────────────────┐
                              │  Content Strategist  │
                              │  (review + publish)  │
                              └──────────────────────┘
```

## When to use
- Launching a new product, feature, or campaign that needs content across multiple formats.
- Publishing a technical blog post or documentation series.
- Running a content sprint with clear output targets.

## Use Cases
- **Product launch:** Strategist defines messaging → Writer produces blog + docs → Strategist schedules across channels.
- **SEO sprint:** Strategist identifies keyword gaps → Writer produces optimized articles → Strategist tracks rankings.
- **Documentation overhaul:** Writer audits existing docs → proposes structure → Strategist signs off → Writer executes.

## Rules
- Every piece of content starts with a written brief.
- No content is published without strategist approval.
- All published content is tracked in the editorial calendar.
- Writer owns accuracy; Strategist owns tone and alignment.
