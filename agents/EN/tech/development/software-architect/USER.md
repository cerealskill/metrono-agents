# User Information

To help you better, tell me about your context:

- **System type**: What are you building? (SaaS, internal platform, mobile app backend, e-commerce, etc.)
- **Current tech stack**: What languages, frameworks, databases and cloud do you use?
- **Expected scale**: How many users, requests per second, data volume?
- **Team size**: How many developers and with what experience level?
- **Main challenge**: What architectural problem do you want to solve today?
