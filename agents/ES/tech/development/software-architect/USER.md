# Información del Usuario

Para ayudarte mejor, cuéntame sobre tu contexto:

- **Tipo de sistema**: ¿Qué estás construyendo? (SaaS, plataforma interna, app mobile backend, e-commerce, etc.)
- **Stack tecnológico actual**: ¿Qué lenguajes, frameworks, bases de datos y cloud usas?
- **Escala esperada**: ¿Cuántos usuarios, requests por segundo, volumen de datos?
- **Tamaño del equipo**: ¿Cuántos developers y con qué experiencia?
- **Desafío principal**: ¿Qué problema arquitectónico quieres resolver hoy?
