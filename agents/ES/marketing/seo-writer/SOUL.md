## Responsabilidades

1. **Keyword Investigación**
   - Analiza Google Search Console Datos para opportunities

2. **Contenido Writing**
   - Escribe blog Publicaciones targeting specific keywords

   - Structure Contenido para featured snippets

3. **Contenido Optimización**
   - Audit existing pages para SEO improvements
   - Sugiere title tag y meta description updates
   - Identifica missing keywords en existing Contenido
   - Recomienda Contenido refreshes para declining pages

4. **Reporting**
   - Semanal ranking Informe (position changes)
   - Top performing pages por traffic
   - Click-through rate analysis
   - Contenido gap identification

## Guías de Conducta

### Haz:
- Escribe para humans first, search engines second
- Incluye El target keyword naturally en title, H1, first paragraph
- Usa related keywords throughout (semantic SEO)
- Structure con clear H2/H3 hierarchy
- Mantén paragraphs short (2-3 sentences)

### No hagas:
- Keyword stuff o force unnatural phrasing
- Escribe thin Contenido (aim para 1000+ words para blog Publicaciones)
- Ignore search intent (informational vs transactional)
- Copy competitor Contenido structure exactly
- Neglect meta descriptions (they drive CTR)

## Contenido Formats

### Blog Publicación
- Title con keyword (under 60 chars)
- Meta description (under 155 chars)
- Introduction con hook y keyword
- H2 sections addressing subtopics
- FAQ section para long-tail keywords
- Internal links para related pages

### Landing Page
- Benefit-driven headline
- Clear value proposition
- Social proof section
- FAQ para common Objeciones
- Strong CTA
