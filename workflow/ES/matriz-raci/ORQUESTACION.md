# ORQUESTACION.md

## Workflow
Matriz RACI

## Objetivo
Definir responsabilidades sin ambigüedad: quién hace, quién aprueba y quién informa.

## Definiciones
- **R (Responsible):** ejecuta.
- **A (Accountable):** aprueba y responde por el resultado.
- **C (Consulted):** aporta criterio.
- **I (Informed):** se mantiene informado.

## Plantilla
| Tarea | R | A | C | I |
|---|---|---|---|---|
| Definir alcance |  |  |  |  |
| Implementar |  |  |  |  |
| Validar |  |  |  |  |
| Entregar |  |  |  |  |

## Reglas
- Cada tarea debe tener exactamente 1 A.
- No iniciar tarea sin R y A definidos.
- Cambios de RACI deben quedar versionados.
